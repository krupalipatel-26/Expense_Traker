# Expense_Traker
A Python-based Expense Tracker that helps users record, manage, and monitor daily expenses. The application allows users to add, view, and calculate total expenses while securely storing data using file handling.

A simple and efficient Expense Tracker built using Python. This application enables users to record their daily expenses, view all saved transactions, and calculate total spending. Expense data is automatically stored in a file, ensuring it is available whenever the application is reopened.

## Features
- Add new expenses
- View all recorded expenses
- Calculate total expenses
- Automatically save expense data
- Load saved expenses on startup
- Simple command-line interface

## Technologies Used
- Python
- File Handling

## Learning Outcomes

- Functions
- Lists and Dictionaries
